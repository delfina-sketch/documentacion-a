# 📌 Calculador de cuotas

### Descripción <a href="#descripcion" id="descripcion"></a>

Este componente permite mostrar en la ficha de productos, el cálculo de cuotas sin interés configurado por monto del producto. Siempre se mostrará la mejor cuota para el precio de ese producto y dentro del modal, las cuotas específicas por promociones bancarias.

<figure><img src="../../.gitbook/assets/4 (12).png" alt=""><figcaption></figcaption></figure>

### Pasos para la configuración

1. Ingresar a **Storefront > Site editor.**&#x20;
2. Para ingresar al bloque, debemos ingresar a una ficha de producto y buscar el bloque llamado **PDP Desktop** y abrirlo. También debemos abrir los bloques llamados **PDP Contenedor Principal Desktop, PDP Info desktop y PDP Cuotas bancarias** e ingresar al bloque llamado **Configuración para las promociones bancarias en PDP**.&#x20;

<figure><img src="../../.gitbook/assets/5 (9).png" alt=""><figcaption></figcaption></figure>

3. Al ingresar el bloque, nos encontraremos con la configuración del bloque:
   1. **Visible?:** Desde esta opción podemos mostrar u ocultar la leyenda del componente.&#x20;
   2. **Categoria a ocultar:** Desde este campo podemos ocultar el componente en alguna categoría de productos.&#x20;
   3. **Promotions icon:** En caso de ser necesario, se puede cargar una imagen para que se visualice como icono de la promoción.&#x20;
   4.  **Máximo de cuotas por monto:** Desde el botón **Agregar** podemos configurar las distintas cuotas a mostrar por monto. <br>

       <figure><img src="../../.gitbook/assets/image (242).png" alt=""><figcaption></figcaption></figure>

       1. Las opciones a crear son las mismas que para PLP:
          1. **Monto mínimo para cuotas:** Debemos completar con el valor mínimo para que comience a aplicar la cuota.&#x20;
          2.  **Cantidad de cuotas:** Debemos completar con la cantidad de cuotas a configurar. <br>

              <figure><img src="../../.gitbook/assets/image (243).png" alt=""><figcaption></figcaption></figure>


   5. Una vez completadas las cuotas, hacemos click en **Aplicar** para que se guarden los cambios.&#x20;

{% hint style="info" %}
Tener en cuenta que esta configuración es para mostrar únicamente las cuotas por monto. \
El resto de las cuotas disponibles por promociones bancarias se visualizarán adentro del modal con el desglose correspondiente.&#x20;
{% endhint %}
